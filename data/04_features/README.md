# data/04_features/ — Text Feature Documentation

## Output files

- `text_features_robusta.csv` — daily text features cho Robusta
- `text_features_arabica.csv` — daily text features cho Arabica

Mỗi file là 1 row per ngày, với cột `date` làm key join với price data.

Generated by: `pipeline/05_disagreement/02_text_features.py`

---

## Pipeline tóm tắt

```
DiscoverLinks.csv (38k rows, TITLE + SNIPPET từ Google Search)
        │
        ├── Date filter: 2023-03-11 → 2026-03-11
        ├── Relevance filter: TITLE hoặc SNIPPET chứa "cà phê"
        ├── Dedup: 1 row per (URL_HASH × TARGET)
        │
        ▼
  Merge với coffee_articles.csv (lấy CONTENT)
        │
        ▼
  Tầng 2: Per-article features (TITLE+SNIPPET → direction/price, CONTENT → body_dir_score)
        │
        ▼
  Tầng 3: Daily aggregate → lag/rolling → discretize SIG_
        │
        ▼
  text_features_{target}.csv
```

## Columns

### Base features (11 cột, aggregate theo ngày)

| Cột | Kiểu | Nguồn | Mô tả |
|-----|-------|-------|-------|
| `date` | datetime | — | Ngày (key join với price data) |
| `n_articles_d` | int | count | Số bài sau relevance filter trong ngày |
| `n_sources_d` | int | count | Số domain (nguồn báo) khác nhau trong ngày |
| `pct_tang_d` | float [0,1] | TITLE+SNIPPET | Tỉ lệ bài có keyword tăng (tăng, tăng vọt, cao hơn) |
| `pct_giam_d` | float [0,1] | TITLE+SNIPPET | Tỉ lệ bài có keyword giảm (giảm, sụt, lao dốc) |
| `pct_on_dinh_d` | float [0,1] | TITLE+SNIPPET | Tỉ lệ bài có keyword ổn định (ổn định, đi ngang, giữ nguyên) |
| `dir_score_ts_d` | float [-1,1] | TITLE+SNIPPET | Media sentiment chính = mean(ts_tang) − mean(ts_giam). Dương = bullish, âm = bearish |
| `dir_score_body_d` | float [-1,1] | CONTENT | Cross-check = mean(body_dir_score). body_dir_score = (n_tăng − n_giảm)/(n_tăng + n_giảm + 1) đếm trên toàn bộ CONTENT |
| `dir_entropy_d` | float ≥ 0 | TITLE+SNIPPET | Shannon entropy của (pct_tang, pct_giam, pct_on_dinh). Cao = ý kiến phân tán, thấp = đồng thuận |
| `price_cv_ts_d` | float ≥ 0 | SNIPPET | Coefficient of Variation giá từ snippets = std/mean. Cao = giá được báo khác nhau nhiều (disagreement) |
| `price_median_ts_d` | float | SNIPPET | Median giá VND/kg trích từ snippets (range hợp lệ: 30,000–250,000) |
| `pct_has_price_ts_d` | float [0,1] | SNIPPET | Tỉ lệ bài có giá cụ thể trong snippet |

### Lag features (24 cột)

Format: `{base_feature}_lag{N}` với N = 1, 2, 3

Lag features được tạo cho 8 base features:

| Base feature | Lý do giữ lag |
|---|---|
| `dir_score_ts_d` | Sentiment chính — xu hướng gần đây |
| `dir_score_body_d` | Cross-check sentiment |
| `pct_tang_d` | Tín hiệu bullish ngày trước |
| `pct_giam_d` | Tín hiệu bearish ngày trước |
| `dir_entropy_d` | Mức đồng thuận ngày trước |
| `price_cv_ts_d` | Disagreement giá ngày trước |
| `n_articles_d` | Mật độ tin ngày trước |
| `n_sources_d` | Đa dạng nguồn ngày trước |

### Rolling features (3 cột)

Format: `{base_feature}_roll3` — rolling mean 3 ngày (shift 1, tức tính từ ngày hôm trước trở về)

Tạo cho: `dir_score_ts_d`, `price_cv_ts_d`, `n_articles_d`

### Discretized signals (12 cột SIG_)

Tất cả là binary (0/1). Thresholds fit riêng cho mỗi target.

| Cột | Logic | Ý nghĩa |
|-----|-------|---------|
| `SIG_MAJORITY_TANG` | pct_tang_d > 0.50 | Đa số bài nói tăng |
| `SIG_MAJORITY_GIAM` | pct_giam_d > 0.50 | Đa số bài nói giảm |
| `SIG_BULLISH` | dir_score_ts_d > 0.15 | Sentiment tổng hợp nghiêng tăng |
| `SIG_BEARISH` | dir_score_ts_d < -0.15 | Sentiment tổng hợp nghiêng giảm |
| `SIG_NEUTRAL` | -0.15 ≤ dir_score_ts_d ≤ 0.15 | Sentiment trung lập |
| `SIG_SPLIT_SIGNAL` | pct_tang > 0.25 AND pct_giam > 0.25 | Tín hiệu xung đột — nguồn nói ngược nhau |
| `SIG_HIGH_DISAGR` | price_cv_ts_d > Q75 | Giá được báo khác nhau nhiều (top 25%) |
| `SIG_LOW_DISAGR` | price_cv_ts_d ≤ Q25 | Giá được báo đồng nhất (bottom 25%) |
| `SIG_HIGH_ENTROPY` | dir_entropy_d > Q75 | Bất đồng quan điểm cao (top 25%) |
| `SIG_MANY_SOURCES` | n_sources_d ≥ 3 | Có ≥ 3 nguồn báo khác nhau |
| `SIG_MANY_ARTICLES` | n_articles_d ≥ median | Số bài nhiều hơn trung vị |
| `SIG_CONFIDENT_DISAGR` | HIGH_DISAGR AND MANY_SOURCES | Disagreement được xác nhận bởi nhiều nguồn |

## Cách sử dụng

### Join với price data

```python
import pandas as pd

price = pd.read_csv("data/06_ground_truth/Investing/arabica_clean.csv")
text  = pd.read_csv("data/04_features/text_features_arabica.csv")

price["date"] = pd.to_datetime(price["date"])
text["date"]  = pd.to_datetime(text["date"])

combined = price.merge(text, on="date", how="left")
```

Lưu ý: Price data chỉ có ngày giao dịch (không weekend/lễ), text features có cả 7 ngày/tuần. Join `how="left"` từ price sẽ tự align.

### Cho Itemset Mining

Dùng các cột `SIG_*` làm transaction items:

```python
sig_cols = [c for c in combined.columns if c.startswith("SIG_")]
transactions = combined[sig_cols].astype(bool)
```

## Lưu ý kỹ thuật

- TITLE + SNIPPET lấy từ Google Search API (DiscoverLinks.csv), không phải từ HTML crawl — sạch hơn rất nhiều so với tách title từ CONTENT.
- Giá chỉ extract từ SNIPPET (regex VND format), KHÔNG extract từ CONTENT (quá nhiễu).
- body_dir_score chỉ đếm keyword tăng/giảm trong CONTENT — chấp nhận noisy, dùng làm cross-check.
- Bài CONTENT < 50 ký tự → body_dir_score = NaN.
- Date range cứng: 2023-03-11 → 2026-03-11 (loại bỏ các ngày 2021, 2022, 2566 do lỗi crawl).
